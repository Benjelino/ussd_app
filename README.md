# **USSD Service for Truck Hauling and Logistic Companies** <br>

# **Description**<br>
A simple ussd application for requesting trucks.
Click [on this link](https://simulator.africastalking.com:1517/) to view the demo.

| USSD APP Features                            |
| --------------------------------------------:| 
| Request to get a call from support           | 
| Request for truck                            |
| Send money from users account to client      |   
| Register                                     |   
| Select Commodity                             |  
| Select route with rate per unit              |  


# Instructions on Demo
- After the link has fully loaded, type any number in the simulator.
- Select USSD from the phone UI in the simulator.
- Enter * 384 * 1554 # into the text input on the sandbox and click send.
- The application starts.
